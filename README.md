# **Student Record System**

**Month & Year:**
July 2023

**Language:**
Python

**End User(s):**
University

## Brief Description:
A software which stores student academic information.

## Software Minimum Requirements:
### Basic Personal Info
* full name, birthdate, pronouns
* contact: phone number, email
* address

### Academic Info
* enrollment date
* major and/or minor
* course history
* grade

### Other
* notes on student
* store on server
